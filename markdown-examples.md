### Ubuntu命令

查看目录 ls

切换目录 cd

创建目录 mkdir 目录名

apt 和 nodejs 的 npm 是一样的（yum是Ubuntu的包管理器）

vim 编辑文件

nginx 安装完成之后 默认放到etc

```sh
apt install git
apt install nginx
apt install nodejs
```
